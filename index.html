Python Variables in Detail

Variables in Python are used to store and manipulate data. They are dynamically typed, meaning you don't need to explicitly declare the type of a variable. The interpreter infers the type based on the value assigned to the variable.

Variable Assignment:

You can assign values to variables using the assignment operator (=). For example: 

        
name = "John"
age = 25
is_student = True

Variable Naming Rules:

Variable names can contain letters, numbers, and underscores. They cannot start with a number. Python is case-sensitive, so age and Age are different variables.

Data Types:

Python has various built-in data types that variables can hold. Common ones include: 

        
# Integer type
age = 25

# Floating-point type
height = 5.9

# String type
name = "John"

# Boolean type
is_student = True

Dynamic Typing:

Python allows dynamic typing, meaning you can reassign a variable to a different type. For example: 

        
x = 5
print(x)  # Outputs: 5

x = "Hello"
print(x)  # Outputs: Hello

Variable Scope:

The scope of a variable determines where it can be accessed. Variables declared inside a function have local scope, while those declared outside functions have global scope. For example: 

        
global_var = "I am global"  # Global variable

def my_function():
    local_var = "I am local"  # Local variable
    print(global_var)  # Accessing global variable
    print(local_var)   # Accessing local variable

my_function()

Python Data Types in Detail

int:

The int data type is used to represent integers (whole numbers) in Python. 

                
        age = 25
        count = -10

float:

The float data type is used to represent floating-point numbers (decimal numbers) in Python. 

                
        height = 5.9
        pi = 3.14159

str:

The str data type is used to represent strings (sequences of characters) in Python. 

                
        name = "John"
        message = 'Hello, Python!'

bool:

The bool data type is used to represent boolean values (True or False) in Python. 

                
        is_student = True
        is_adult = False

list:

The list data type is used to represent ordered, mutable sequences of elements in Python. 

                
        numbers = [1, 2, 3, 4, 5]
        names = ["Alice", "Bob", "Charlie"]

tuple:

The tuple data type is used to represent ordered, immutable sequences of elements in Python. 

                
        coordinates = (3, 4)
        colors = ("red", "green", "blue")

set:

The set data type is used to represent unordered, mutable collections of unique elements in Python. 

                
        unique_numbers = {1, 2, 3, 4, 5}
        unique_colors = {"red", "green", "blue"}

dict:

The dict data type is used to represent key-value pairs (associative arrays) in Python. 

                
        person = {"name": "John", "age": 25, "occupation": "Developer"}
        student_grades = {"math": 90, "english": 85, "science": 92}

Python Operators in Detail

Arithmetic Operators:

Arithmetic operators are used to perform mathematical operations in Python. 

        
# Addition
result = 5 + 3

# Subtraction
difference = 10 - 7

# Multiplication
product = 4 * 6

# Division
quotient = 15 / 3

# Modulus (remainder)
remainder = 17 % 5

# Exponentiation
power = 2 ** 3

Comparison Operators:

Comparison operators are used to compare values in Python, resulting in a boolean value (True or False). 

        
# Equal to
is_equal = 10 == 5

# Not equal to
not_equal = 8 != 8

# Greater than
greater_than = 15 > 10

# Less than
less_than = 20 < 25

# Greater than or equal to
greater_equal = 12 >= 12

# Less than or equal to
less_equal = 30 <= 35

Logical Operators:

Logical operators are used to combine or manipulate boolean values. 

        
# AND
logical_and = True and False

# OR
logical_or = True or False

# NOT
logical_not = not True

Assignment Operators:

Assignment operators are used to assign values to variables. 

        
# Assign value
x = 10

# Add and assign
x += 5  # Equivalent to x = x + 5

# Subtract and assign
y -= 3  # Equivalent to y = y - 3

# Multiply and assign
z *= 2  # Equivalent to z = z * 2

# Divide and assign
w /= 4  # Equivalent to w = w / 4
